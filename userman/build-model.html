Building the Static Object Model

You need to rebuild the static object model if there are any changes to the raw data (any of the XML files in the data directory). You also need to rebuild the static object model if you make any changes to the structure or contents of the persistent static object model.

To do a full build of the static object model, use the following command:


% full-build

The full build process and the formats of the various raw data XML files are described in more detail in the section on Adding New Texts.
