Saved Word Queries

Introduction to Saved Queries

WordHoard allows you to create saved queries using its own internal query language. You can use saved word queries to create word sets, and saved bibliographic queries to create work sets. The syntax for the query language was not originally intended to be exposed for public use, so it lacks "user friendliness." Future versions of WordHoard will allow creation and editing of the queries using a visual interface. You will probably find that visually constructed ad-hoc queries for searching for words and searching for works sufficient and easier to use than saved queries.

To create, edit, or delete saved queries you must first login so that the Sets menu becomes available. Not all installations of WordHoard allow you to login. You will not be able to create saved queries on such systems.
