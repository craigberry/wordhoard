Parts of Speech XML File

The file pos.xml defines the parts of speech. The standard NU file is:

<?xml version="1.0" encoding="utf-8"?>
<WordHoardPos>
   <pos id="("
      syntax="pu"
      tense=""
      mood=""
      voice=""
      case=""
      gender=""
      person=""
      number=""
      degree=""
      negative=""
      language="english"
      wordClass="pu"/>
   ...
   <pos id="zz"
      syntax="zz"
      tense=""
      mood=""
      voice=""
      case=""
      gender=""
      person=""
      number=""
      degree=""
      negative=""
      language="english"
      wordClass="zz"/>
</WordHoardPos>

The parts of speech definition file has the following elements:

  • WordHoardPos. The root element.

    Children:

    • pos (0..n). Part of speech definitions.
  • pos. Part of speech definition.

    Attributes:

    • id. Required part of speech id.
    • syntax. Optional syntax.
    • tense. Optional tense.
    • mood. Optional mood.
    • voice. Optional voice.
    • case. Optional case.
    • gender. Optional gender.
    • person. Optional person.
    • number. Optional number.
    • degree. Optional degree.
    • negative. Optional negative.
    • language. Language: "english" or "greek".
    • wordClass. Optional word class id.

    Children:

    • TEXT (0..1). Description.
