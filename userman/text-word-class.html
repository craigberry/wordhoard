The Word Classes XML File

The file word-classes.xml defines all of the word classes. The standard NU file is:

<?xml version="1.0" encoding="utf-8"?>
<WordHoardWordClasses>
   <wordClass id="j"
      majorClass="adjective">adjective</wordClass>
   <wordClass id="jn"
      majorClass="adjective">adjective/noun</wordClass>
   ...
   <wordClass id="pd"
      majorClass="pronoun">demonstrative pronoun</wordClass>
   <wordClass id="it"
      majorClass="foreign word">Italian</wordClass>
</WordHoardWordClasses>

The word classes definition file has the following elements:

  • WordHoardWordClasses. The root element.

    Children:

    • wordClass (0..n). Word class definitions.
  • wordClass. Word class definition.

    Attributes:

    • id. Required word class id.
    • majorClass. Required major word class.

    Children:

    • TEXT (0..1). Description.
